## 📊 Complexity Reference Guide

### Algorithms
| Algorithm | Time | Space | Use Case |
|-----------|------|-------|----------|
| Binary Search | O(log n) | O(1) | Sorted data lookup |
| Quick Sort | O(n log n) avg | O(log n) | General sorting |
| Merge Sort | O(n log n) | O(n) | Guaranteed performance |
| BFS | O(V+E) | O(V) | Shortest path (unweighted) |
| DFS | O(V+E) | O(V) | Topology, connectivity |
| Dijkstra | O((V+E)logV) | O(V) | Shortest path (weighted) |
| DP | Varies | Varies | Optimization problems |
| Greedy | Varies | O(1) | Scheduling, optimization |
| Backtracking | O(N!) worst | O(N) | Constraint satisfaction |
| Hashing | O(1) avg | O(n) | Fast lookup |

### Data Structures
| Structure | Insert | Search | Space | Use Case |
|-----------|--------|--------|-------|----------|
| Array | O(n) | O(n) | O(n) | Fixed size data |
| Linked List | O(1) | O(n) | O(n) | Dynamic lists |
| Stack | O(1) | O(n) | O(n) | LIFO operations |
| Queue | O(1) | O(n) | O(n) | FIFO scheduling |
| Hash Table | O(1) avg | O(1) avg | O(n) | Fast lookup |
| BST | O(log n) avg | O(log n) avg | O(n) | Sorted access |
| Heap | O(log n) | O(1) peek | O(n) | Priority queue |
| Graph | O(1) | O(degree) | O(V+E) | Network topology |
| Trie | O(m) | O(m) | O(k*m) | Prefix matching |
| Bloom | O(k) | O(k) | O(m) bits | Membership test |

                 DATABASE ORACLE IN C - START HERE

Welcome to the Database Oracle Learning Project!

This is a comprehensive 7-phase learning path for building a mini Oracle-like
database system in C. You'll learn how real databases work by building each
component from scratch.

================================================================================
                            WHAT YOU'LL BUILD
================================================================================

Phase 1: Key-Value Store ✅
  - Basic key-value data structure
  - File persistence
  - Serialization

Phase 2: B+ Tree Index ✅
  - B+ tree for efficient searching
  - Page management
  - Node splitting

Phase 3: SQL Parser ✅
  - Lexer for tokenization
  - Parser for syntax analysis
  - AST construction

Phase 4: Buffer Pool ✅
  - In-memory page cache
  - LRU replacement policy
  - Dirty page tracking

Phase 5: Transactions (Coming Soon)
  - Write Ahead Logging (WAL)
  - ACID properties
  - Crash recovery

Phase 6: Concurrency Control (Coming Soon)
  - Lock manager
  - MVCC
  - Deadlock detection

Phase 7: Client-Server (Coming Soon)
  - TCP server
  - Network protocol
  - Connection pooling

================================================================================
                            COMMON ISSUES
================================================================================

Issue: "make: command not found"
Solution: Install make (apt-get install make on Linux)

Issue: "gcc: command not found"
Solution: Install GCC (apt-get install gcc on Linux)

Issue: Tests fail
Solution: Check error messages, read DEBUGGING_GUIDE.md

Issue: Memory leaks
Solution: Run "make valgrind" to find leaks

Issue: Segmentation fault
Solution: Run "make gdb" and use backtrace command

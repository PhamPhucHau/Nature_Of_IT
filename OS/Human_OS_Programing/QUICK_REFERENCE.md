# ⚡ Human OS Programming - Quick Reference

## One-Page Quick Start

### What Is This?
A 12-week course to build a **Personal Operating System** in C that helps you manage your life like a computer manages processes.

### Core Idea
```
Your Life = Operating System
├─ Tasks = Processes
├─ Energy = CPU Resources
├─ Time = CPU Time
├─ Knowledge = Memory
└─ Goals = System Objectives
```

## Key Concepts

### Task Management
```c
Task = {
    id, name, priority,
    duration, deadline,
    energy_required, status,
    dependencies
}
```

### Energy Management
```c
Energy = {
    mental_energy,      // Thinking
    physical_energy,    // Movement
    emotional_energy,   // Relationships
    total_energy        // Sum
}
```

### Decision Making
```c
Decision = {
    options[],
    criteria[],
    weights[],
    chosen_option
}
```

### Learning
```c
Experience = {
    task_id, decision_id,
    outcome, satisfaction,
    timestamp, notes,
    energy_used, time_spent
}
```

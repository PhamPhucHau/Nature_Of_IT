HTTP Server - Phase 2

Status

✓ Server Running

Features Implemented

  • HTTP Request Parsing - Parses method, path, version, and headers
  • HTTP Response Building - Builds proper HTTP responses with headers
  • Static File Serving - Serves HTML, CSS, JS, and image files
  • Content-Type Detection - Automatically detects file types
  • 404 Error Handling - Returns proper 404 responses
  • Multi-threaded - Handles multiple clients concurrently

Testing

Test the server with these commands:

# Start the server
make run

# Test with curl
curl http://localhost:8080/
curl http://localhost:8080/404.html

# Test different content types
curl -H "Accept: text/html" http://localhost:8080/
curl -H "Accept: application/json" http://localhost:8080/api

HTTP Methods Supported

  • GET - Retrieve resources
  • POST - Submit data
  • PUT - Update resources
  • DELETE - Remove resources
  • HEAD - Get headers only
  • OPTIONS - Get options
  • PATCH - Partial updates

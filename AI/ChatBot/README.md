# ChatBot Product - Complete Implementation
## Local LLM ChatBot with Ollama, Spring Boot & React

**Status:** ✅ Production-Ready Architecture  
**Last Updated:** June 23, 2026

---

## 📋 Overview

A full-stack ChatBot application with:
- **Frontend:** React with modern UI/UX
- **Backend:** Spring Boot with clean architecture layers
- **LLM:** Ollama (local, no cloud dependency)
- **Database:** PostgreSQL + H2 (for dev)
- **Tracing:** Global request ID in all logs for easy debugging
- **Middleware:** Anylink request interceptor for distributed tracing

---

## 🎯 Key Features

✅ **Model Selection:** Switch between Llama 2, Mistral, Neural Chat, etc.  
✅ **Chat History:** Full conversation persistence  
✅ **Request Tracing:** Global request ID in every log entry  
✅ **Clean Architecture:** Proper layer separation  
✅ **Error Handling:** Centralized exception handling  
✅ **Security:** CORS, input validation, secure headers  
✅ **Local First:** Runs entirely locally (no external API calls)  
✅ **Production Ready:** Docker, load balancing, monitoring ready  

---

## 🏗️ Architecture

```
Frontend (React)
    ↓
Anylink Middleware (Global Request ID)
    ↓
Spring Boot Backend (Controller → Service → Repository)
    ↓
Ollama LLM Server
    ↓
PostgreSQL Database
```

**All logs include the Global Request ID for complete traceability:**

```
[req_a1b2c3d4e5f6] INFO - Incoming request: POST /api/chat/send
[req_a1b2c3d4e5f6] INFO - ChatController.send() invoked
[req_a1b2c3d4e5f6] INFO - OllamaService.generate() called
[req_a1b2c3d4e5f6] INFO - Response sent to client
```

## 🚀 Quick Start

### Prerequisites
- Docker & Docker Compose
- (Optional: Java 17+, Node.js 18+, for local development)

### Start Everything (5 minutes)

```bash
# Clone the repository
git clone <your-repo>
cd chatbot

# Start all services (Ollama, Backend, Database)
docker-compose up -d

# Pull an LLM model
docker exec ollama-server ollama pull llama2

# Start frontend (in another terminal)
cd frontend
npm install
npm run dev
```

**Done!** Access the application at `http://localhost:3000`

## 🔌 API Endpoints

### Send Message
```
POST /api/chat/send
Content-Type: application/json
X-Request-ID: <auto-generated>

{
  "message": "What is AI?",
  "model": "llama2",
  "conversationId": "conv-123"
}

Response:
{
  "response": "AI is...",
  "model": "llama2",
  "conversationId": "conv-123",
  "responseTimeMs": 2500,
  "requestId": "req_a1b2c3d4e5f6"
}
```

### Get Chat History
```
GET /api/chat/history/{conversationId}
X-Request-ID: <auto-generated>

Response: [
  {
    "response": "...",
    "model": "llama2",
    "conversationId": "conv-123"
  }
]
```

### Clear Conversation
```
DELETE /api/chat/history/{conversationId}
X-Request-ID: <auto-generated>
```

## 📊 Request Flow Example

```
1. User types: "What is machine learning?"
2. Frontend generates request ID: req_a1b2c3d4e5f6
3. Sends to Backend with X-Request-ID header
4. RequestIdFilter extracts ID and stores in ThreadLocal
5. ChatController logs: [req_a1b2c3d4e5f6] Processing message
6. ChatService logs: [req_a1b2c3d4e5f6] Calling Ollama
7. OllamaService logs: [req_a1b2c3d4e5f6] Response in 2.5s
8. Response saved to DB with request ID
9. Response sent back to Frontend with request ID
10. All logs can be traced with single request ID
```

## 🚢 Deployment

### Development
```bash
# Backend
mvn spring-boot:run

npm run dev

ollama serve
```

### Production (Docker)
```bash
docker-compose -f docker-compose.yml up -d
```

### Scaling
- Horizontal: Run multiple backend instances behind load balancer
- Vertical: Increase Java heap size, PostgreSQL resources

## 🐛 Troubleshooting

### Backend won't connect to Ollama
```bash
# Check Ollama is running
curl http://localhost:11434/api/tags

# Check logs
docker-compose logs ollama
```

### Frontend can't reach backend
```bash
# Check CORS is enabled in backend
# Check backend is running on 8080
curl http://localhost:8080/actuator/health
```

### Models not available
```bash
# Pull models
docker exec ollama-server ollama pull llama2
docker exec ollama-server ollama pull mistral
```

🚀 5-MINUTE QUICK START:

Step 1: Start Services
  $ docker-compose up -d

Step 2: Pull LLM Model
  $ docker exec ollama-server ollama pull llama2

Step 3: Start Frontend
  $ cd frontend && npm install && npm run dev

Step 4: Open Browser
  → http://localhost:3000

Step 5: Start Chatting!

📁 ARCHITECTURE:

Frontend (React)
    ↓
Anylink Middleware (Global Request ID)
    ↓
Spring Boot Backend
    ├─ Controller: ChatController
    ├─ Service: ChatService, OllamaService
    ├─ Repository: ChatHistoryRepository
    └─ Security: RequestIdFilter, CorsMvcConfig
    ↓
Ollama LLM Server (localhost:11434)
    ↓
PostgreSQL Database

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔌 API ENDPOINTS:

POST /api/chat/send
  Send message to chatbot

GET /api/chat/history/{conversationId}
  Get chat history

DELETE /api/chat/history/{conversationId}
  Clear conversation

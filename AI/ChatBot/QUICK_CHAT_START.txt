================================================================================
                       CHATBOT - QUICK START GUIDE
================================================================================

🎯 JUST WANT TO CHAT? HERE'S WHAT TO DO:

1. Open Browser
   URL: http://localhost:3000

2. You'll See
   "Welcome to ChatBot!"
   "Click ➕ New to start a conversation"

3. Click This Button
   ➕ "Start New Conversation" (the big purple button)
   OR click ➕ "New" in top right

4. Message Input Activates
   The input field (at the bottom) becomes enabled
   It was gray before, now it's white/active

5. Start Typing
   Click in the input field
   Type your message or question
   Example: "Hello!" or "What is AI?"

6. Send Message
   Press Enter or click 📤 Send button

7. See AI Response
   Your message appears as "USER"
   AI response appears as "ASSISTANT"
   You can continue the conversation

8. More Features
   • Create multiple conversations
   • Switch between them in the sidebar
   • Delete conversations with 🗑️
   • Change AI model at top
   • View full message history

================================================================================
                            TROUBLESHOOTING
================================================================================

PROBLEM: Input field is grayed out
SOLUTION: Click ➕ "New" button to create conversation

PROBLEM: No input field at all
SOLUTION: Refresh page (Ctrl+Shift+R)

PROBLEM: Messages won't send
SOLUTION: 
  1. Create a conversation first
  2. Make sure input field is white (not gray)
  3. Type something in the field
  4. Press Enter or click Send

PROBLEM: Page won't load
SOLUTION: Check URL is exactly: http://localhost:3000

PROBLEM: Nothing appears on screen
SOLUTION: 
  1. Check browser console (F12)
  2. Check if backend is running
  3. Try refreshing page
  4. Clear browser cache

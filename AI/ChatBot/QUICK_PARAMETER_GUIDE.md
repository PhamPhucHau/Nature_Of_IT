# Quick Parameter Guide

**TL;DR - Model Parameter Settings Feature**

---

## 🎯 What's New?

Click **⚙️ Show Parameters** to adjust how the AI generates responses!

---

## 🎨 Quick Presets (One-Click)

### Creative
Use for: Stories, poetry, brainstorming  
Effect: More diverse, imaginative responses

### Balanced (Default)
Use for: General chat, everyday questions  
Effect: Good quality and natural flow

### Precise  
Use for: Technical questions, factual answers  
Effect: Focused, accurate responses

### Deterministic
Use for: Testing, reproducibility  
Effect: Consistent, same answer every time

---

## 🎚️ Main Parameters (Sliders)

### Temperature (0.0 - 2.0)
**What it does:** Controls randomness  
**Try:** 
- Low (0-0.5): Serious, focused answers
- Medium (0.7): Balanced (default)
- High (1.5+): Very creative, wild responses

### Top-P (0.0 - 1.0)
**What it does:** Limits which words are considered  
**Simple:** Higher = more variety

### Top-K (1 - 100)
**What it does:** How many word choices at each step  
**Simple:** Higher = more options = more variety

### Max Tokens (100 - 4096)
**What it does:** How long the answer can be  
**Simple:** Lower = shorter, Higher = longer

### Repetition Penalty (1.0 - 2.0)
**What it does:** Discourages repeating words  
**Simple:** Higher = less repetition

## 🔧 How to Use

1. **Click** "⚙️ Show Parameters" button
2. **Choose** a preset OR adjust sliders
3. **Send** a message - parameters apply automatically
4. **Repeat** - different presets for different questions

---

## 💡 Tips

- **Start with Balanced** if unsure
- **Try Creative** for writing/brainstorming  
- **Use Precise** for homework/technical help
- **Use Deterministic** for reproducible testing
- **Mix & Match** - use preset then tweak sliders

---

## ❓ Common Questions

**Q: My changes didn't work?**  
A: Changes apply to the NEXT message, not the one before

**Q: What's the best setting?**  
A: Start with "Balanced" preset - it works for most things

**Q: Can I use multiple settings?**  
A: Yes! Use a preset first, then adjust individual sliders

**Q: Will parameters affect all my conversations?**  
A: No - each message uses current settings. New message = apply settings again

**Q: Is there a "wrong" setting?**  
A: No - all ranges are safe. Experiment and find what you like!

---

## 🚀 Advanced Tips

**For Code Help:**
1. Click "Precise" preset
2. Reduce Max Tokens to 1024 for quick snippets
3. Increase for full implementations

**For Creative Writing:**
1. Click "Creative" preset  
2. Increase Temperature to 1.3+
3. Increase Max Tokens to 3000+

**For Exact Same Answers:**
1. Click "Deterministic" preset
2. Set a Seed number if needed
